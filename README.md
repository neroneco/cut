## Instalation
* Software requirements:
    - `gcc` or `clang` compiler
    - `make` build system
    - `CC` environment variable must be set to used compiler
* Commands:
  - `make` - compile all files
  - `make dbg` - compile with debug symbols
  - `make test` - compile and run tests
